GSI VETAR
---------

This project is compound of projects:

1) wr_core_demo/vetar_top.qsf -> Altera Platform
2) cpld_vetar                 -> Xilinx Platform

in order to get both gateware, please from the gsi_starter_kit, run:

make vetar.

More documentation about how to flash a card, build from sources... :
http://www.ohwr.org/projects/gsi-timing-starter-kit/wiki
